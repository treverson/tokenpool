Getting Started

Mining software

1. NVIDIA Cuda open source (@Mikers/@Azlehria) (requires Nvidia 391.01 drivers and Nvidia CUDA 9.1)

Download CUDA Miner (Win/Linux) from Github

* Use version 2.10 for Windows 10 and version 2.9 for Windows 7.


Run miner '0xbtcminer-win.exe'
Type 'help' for all commands
For new miners: new account: 'account new'. Enter new/your password and remember it
* Current miners can use the same ETH address they used before or create new address To view and copy wallet address/private key: 'account list'.
***Alwasys save private keys in a safe place and do not share them!
Set the miner to mine with this pool 'pool select http://0xBTCpool.com:8586'
Start mining with 'pool mine cuda' for GPU
2. AMD OpenCL MViS (2% devfee) (Win/Linux) (works with most NVIDIA cards too)

Download OpenCL Miner (Win/Linux) from Github

Developer installation/setup instructions

  • Move tokenminer.ini to %LocalAppData%/tokenminer on Windows, or $HOME/.config/tokenminer on Linux. * If that folder path does not exist, you will need to create it manually.
  • For POOL MINING, the main thing you need to specify in the .INI file is your ETH account address to which rewards will be paid out.

    Edit tokenminer.ini
    Change "Host= " to "Host=http://0xBTCpool.com" and "RPCPort=" to "RPCPort=8586"
    Add your 0xBTC address to "MinerAcct="0x............................... *replace 0x............ with your address
    Save tokenminer.ini *** Do not add your Privte Key.
    To run the miner Open CMD/Command and navigate to the folder you saved the miner to
    Run 'tokenminer.exe --list-devices -G' to find your GPU and the platform #
    Run 'tokenminer.exe -P -G --opencl-platform n' n= your GPU platform #.
