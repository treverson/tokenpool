Getting Started

Mining software

1. NVIDIA Cuda open source (@Mikers/@Azlehria) or Mag517 COSMIC miner. (requires Nvidia 391.01 drivers and Nvidia CUDA 9.1)

Download CUDA Miner
Download COSMIC v3 Miner


  • Run miner '0xbtcminer-win.exe' or 'Cosmic-0xBTC-HB_V3.4t-win64.exe'
  • Type 'help' for all commands
  • New miners: To create new ETH account type: 'account new'. Enter new/your password and remember it
  • If you want to use your current ETH account address, type: 'account select 0x.....................' '(replace 0x.... with your ETH account)'
    * Current miners can use the same ETH address they used before or create new address
  • To view and copy wallet address/private key: 'account list'. ***Alwasys save private keys in a safe place and do not share them!
  • Set the miner to mine with this pool 'pool select http://0xBTCpool.com:8586'
  • Start mining with 'pool mine cuda' for GPU

    Developer installation/setup instructions

  • Edit tokenminer.ini
  • Change "Host= " to "Host=http://0xBTCpool.com" and "RPCPort=" to "RPCPort=8586"
  • Add your 0xBTC address to "MinerAcct="0x............................... *replace 0x............ with your ETH address
  • Remove ';' before the line with 'MinutesPerShare=2' and change it to 'MinutesPerShare=pool'
  • Save tokenminer.ini *** Do not add your Privte Key.
  • To run the miner Open CMD/Command and navigate to the folder you saved the miner to
  • Run 'tokenminer.exe --list-devices -G' to find your GPU and the platform #
  • Edit 'star-mining.bat' with the platform # of your GPU'--opencl-platform n' n= your GPU platform #. And save
  • Run 'star-mining.bat'
